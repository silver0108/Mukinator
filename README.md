# Mukinator
## 서비스 소개
음식의 다양한 특성 ( 한식-중식-양식 등 / 따뜻한- 차가운 / 면 - 밥 - 빵 / 등등) 을 기반으로 사용자가 먹고 싶은 음식을 안내해주는 서비스

<strong>1. 음식을 골라 주는 기능
<br/>
<strong>2. 지역별 맛집 후기 게시판 기능
